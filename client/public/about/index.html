About The Sharks Softball Dashboard

By The Sharks PCLL Softball Coaching Staff · Updated April 2026

The Sharks Softball Dashboard is the official game-day coaching tool for The Sharks, a 12U Girls Softball team competing in the Palm Coast Little League (PCLL) Spring 2026 season in Palm Coast, Florida.

Purpose

Built for dugout use on mobile devices, this dashboard enables coaches to make data-driven decisions quickly during live games. The system ingests game statistics from GameChanger CSV exports and processes them through a Python analytics pipeline to generate lineup recommendations, opponent insights, and practice plans.

Features

  • Lineup Optimizer — Intelligent batting order recommendations based on player availability, batting averages, and historical performance across the season.
  • SWOT Analysis — Opponent scouting with strengths, weaknesses, opportunities, and threats analysis for each rival team in the Palm Coast Little League division.
  • Game Statistics — Comprehensive batting averages, on-base percentages, slugging percentages, and fielding statistics for all Sharks players across the full season.
  • Practice Generator — Customized practice plans targeting weaknesses identified from recent game performance and season-long statistical trends.
  • Apex Announcer — A walk-up music and TTS player introduction system powered by the Web Audio API for exciting game-day announcements from the PA system.
  • Player Availability — Per-game availability tracking that automatically integrates into lineup and batting order recommendations.

Technology

The dashboard is built with React on the frontend and a Python Flask API on the backend. It is served by nginx from a Raspberry Pi 5 hosted in Palm Coast, Florida, with continuous deployment via GitHub Actions and Watchtower. Statistics are processed from GameChanger CSV exports and optionally supplemented with scorebook OCR for deeper analysis.

About The Team

The Sharks compete in the Palm Coast Little League, located in Palm Coast, Florida. The Spring 2026 season features games against division rivals including the Ravens, Peppers, and Riptide Rebels at PCLL fields throughout Palm Coast. The team competes in the 12U Girls Softball division.

This dashboard is a private coaching tool and is not intended for general public use. For more information about Palm Coast Little League, contact the league through their official organization channels.
